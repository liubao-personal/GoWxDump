
## 特别说明
本项目是基于[https://github.com/SpenserCai/GoWxDump](https://github.com/SpenserCai/GoWxDump)而来

### 交互式命令
```bash
show_info 获取微信基础信息
decrypt 解密数据
friends_list 获取好友列表 （目前支持：获取最近十个聊天的好友信息，需要解密后才能获取）
```
### 非交互式命令
```bash
GoWxDump.exe -spy
```
